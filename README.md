# AutotunePlugIn
Vocal Autotune plugin for DAW's 

The Fundamental Process (Abstracted):

Detects the main frequency(pitch) of the singer's note ----> decides what the “correct” note should be (for example the nearest note in a scale) ----> bends the original sound to match that note.


1. **Pitch detection**

   I used the popular YIN-algorithm.
